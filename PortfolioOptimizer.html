Project overview

PortfolioOptimizer is a comprehensive Python application that applies Modern Portfolio Theory to construct mathematically optimal investment portfolios. It uses historical stock data to calculate the efficient frontier and provides robust backtesting tools to validate model performance across different time periods.

Core Features

  • Mean-variance optimization using Markowitz portfolio theory
  • Efficient frontier visualization with risk-return tradeoffs
  • Real-world backtesting across different time periods
  • Monte Carlo simulation for portfolio performance modeling
  • Yahoo Finance API integration for live market data
  • Interactive command-line interface with data validation

Mathematical Foundation

The program implements Lagrangian optimization to solve the constrained minimization problem for portfolio variance. It calculates covariance matrices, constructs the global minimum variance portfolio, and generates efficient portfolios for user-specified return levels.

Key Workflow

1. Data Collection & Validation

Enter stock tickers and time period. The system fetches historical data via Yahoo Finance API and handles missing data scenarios by allowing users to either skip periods or remove problematic tickers.

2. Efficient Frontier Generation

Calculate and visualize the efficient frontier showing optimal risk-return combinations. Display the global minimum variance portfolio and allow selection of target return levels.

3. Backtesting Options

Choose between real-world backtesting (apply optimized weights to different time periods) or Monte Carlo simulation (generate synthetic performance scenarios).

How to run

Setup and execution steps:

# 1. Clone the repository
git clone https://github.com/aarwitz/PortfolioOptimizer.git
cd PortfolioOptimizer

# 2. Install dependencies
pip install pandas numpy matplotlib yfinance

# 3. Run the main program
python -c "from find_optimized_portfolio import main; main()"

# Alternative: open Python REPL and run
python
>>> from find_optimized_portfolio import main
>>> main()

The program provides an interactive command-line interface that guides you through ticker selection, date ranges, and analysis options. All visualizations display in separate matplotlib windows.

Architecture & Files

  • find_optimized_portfolio.py — main entry point and workflow orchestration
  • matrix_manip.py — core mathematical operations, Lagrangian optimization
  • DF_helper.py — data fetching, processing, and Yahoo Finance integration
  • RealTest.py — real-world backtesting engine and performance validation
  • MonteCarlo.py — Monte Carlo simulation class for synthetic modeling
  • plots.py — efficient frontier and backtesting visualization
  • user.py — interactive CLI and user input handling
  • prints.py — formatted output and results display

Technical Highlights
