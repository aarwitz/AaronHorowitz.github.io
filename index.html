First Steps with GitHub Pages

Published on September 9, 2025

Hello, world! Welcome to my new blog. This is my very first post, created to get my GitHub Pages site up and running. It's amazing how simple it is to publish a personal site or blog directly from a repository.

The process is straightforward: just add your HTML and CSS files, set the source branch in your repository settings, and a moment later, your page is live. I'm excited to share more content here soon. Stay tuned!
