Project overview

Robot Configuration Space is a comprehensive Python framework for robotics motion planning that implements collision detection and path optimization for two-link manipulators. The system combines computational geometry, artificial potential fields, and gradient descent algorithms to enable safe navigation in obstacle-rich environments.

Core Features

  • Two-link manipulator forward kinematics and collision detection
  • Artificial potential field generation with attractive and repulsive components
  • Gradient descent motion planning with configurable parameters
  • Real-time visualization of robot configurations and collision states
  • Polygon-based obstacle representation with visibility algorithms
  • Spherical obstacle modeling with distance influence fields

Technical Implementation

The framework models robots as sets of polygons and obstacles as star-shaped boundaries or spherical influences. Robot edges are colored red when in collision, green otherwise. The motion planning algorithm uses artificial potential functions to guide the manipulator toward goals while avoiding obstacles through repulsive fields.

Algorithm Details

1. Collision Detection

Implements polygon-based collision checking using visibility algorithms and edge intersection tests. Supports both filled and hollow obstacle representations.

2. Potential Field Generation

Creates attractive potentials toward goal positions and repulsive fields around obstacles. Supports both conic (linear) and quadratic potential functions for improved convergence.

3. Gradient Descent Optimization

Implements iterative motion planning using potential field gradients. Configurable step size and convergence criteria ensure robust path generation.

How to run

Setup and execution steps:

# 1. Clone the repository
git clone https://github.com/aarwitz/Robot-Configuration-Space.git
cd Robot-Configuration-Space

# 2. Install dependencies
pip install numpy scipy matplotlib

# 3. Run collision detection demo
python me570_hw3.py

# 4. Execute motion planning
python -c "from me570_potential import Planner; Planner().run_plot()"

# 5. Test specific modules
python -c "import me570_geometry; me570_geometry.field_plot_threshold(lambda x: x[0]**2 + x[1]**2)"

The system includes comprehensive test functions that demonstrate collision detection, potential field visualization, and motion planning results. All visualizations use matplotlib for real-time display.

Architecture & Files

  • me570_robot.py — two-link manipulator kinematics, collision detection, and visualization
  • me570_geometry.py — computational geometry classes (Polygon, Sphere, Edge, Grid, Torus)
  • me570_potential.py — artificial potential fields, motion planner, and sphere world loader
  • me570_hw3.py — demonstration functions and test cases for collision detection
  • sphereworld.mat — MATLAB data file containing obstacle world definition
  • twolink_testData.mat — test points for collision detection validation
  • pygments_homework3.py — HTML code documentation generator
